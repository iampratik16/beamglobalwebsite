# Client logos

Monochrome treatment is applied in CSS (greyscale + reduced opacity, colour on hover),
so drop in normal-colour logos here. Transparent-background SVG/PNG looks best.

To swap a logo: replace the file with the same name — no code changes needed.
Filenames are referenced from `src/data/clients.ts`.
